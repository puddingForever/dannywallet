# 가계부 어플리케이션 Danny Wallet (1人 프로젝트)

### 프로젝트 이름 : Danny Wallet 
### 프로젝트 기간 : 2023-05-10 ~ 
### 주요 기능 
* 월별 예산 등록 
* 소비 유형마다 카테고리 세분화
  * 집=[ 전기세 , 수도세 , 월세 ] , 식료품=[ 친구선물 , 옷 , 식료품 ]
* 카테고리마다 예산 제한
* 총 월 사용금액은 다음달 예산안에 반영됨 
  * 월 예산안보다 더 사용시 다음달 예산에서 마이너스 , 월 예산안보다 적게 사용시 다음달 예산에서 플러스 
* 소비 카테고리마다 서로 잔돈을 보낼 수 있음
